## Grade Predictor for Students

A simple machine learning project to predict student performance using Python and basic data analysis.## About The Project

This project is a prediction of whether a student may pass or fail based on:

- Daily study hours
- attendance %
- Latest score

It’s a basic machine learning model trained on sample student data.# Features 

- Predict pass or failure
- Used Python
- Uses simple machine learning
- Project for beginners
- Basic input for terminals
- Good 1st data science project ### Technologies Used

 - Python 
 - Pandas 
- Scikit-learn ## Running

1. Install Python on your machine.
2. Open your project directory.
3. Install required libraries:
bash pip install requirements.txt
